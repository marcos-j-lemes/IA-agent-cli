# IA-agent-cli
Esse repositorio é de uma agente


How to rn

```bash
pip install -r requirements.txt
python main.py                              # REPL
python main.py "find all .env files"       # single prediction
python main.py --dataset my_data.json      # train on your data
python main.py --dataset my_data.json --save model.joblib  # save trained model
```
